# Laptop Provisioning Process

**Owner:** IT Operations
**Last updated:** 2025-12-01
**Applies to:** All new hires and existing employees needing a refresh

## Standard provisioning

New hires are provisioned a laptop based on role:

- **Engineering roles**: MacBook Pro 14", 32GB RAM, configured with
  standard dev tooling pre-installed (see Coding Standards docs for what
  this includes per language).
- **Non-engineering roles**: MacBook Air 13", 16GB RAM.
- **Robotics lab / hardware roles**: additionally receive a
  Linux-based workstation for on-robot development, provisioned
  separately through the Robotics IT liaison rather than standard IT.

## Timeline

Provisioning is triggered as soon as a start date is confirmed in the HR
system, which should be well before day one — see Onboarding Checklist:
New Hires. Standard lead time is 5-7 business days for shipping
(remote/hybrid hires) or desk delivery (office-first hires). Rush
provisioning (2-3 days) is available for late-confirmed start dates by
flagging the request as urgent in the IT portal.

## What's pre-installed

All laptops ship with NorthLink VPN client (see VPN Setup Guide), MFA
enrollment tools, standard productivity suite, and endpoint security
monitoring required under Device Security Requirements. Engineering
laptops additionally have git, Docker, and language-specific toolchains
pre-installed per the relevant coding standards document.

## Refresh cycle

Laptops are refreshed every 3 years, or sooner if there's a
performance-impacting hardware issue. To request an early refresh outside
the standard cycle, file a ticket with justification; these are evaluated
case by case rather than automatically approved.

## Loss, theft, or damage

Report immediately to IT Helpdesk and your manager — do not wait for a
convenient time. Lost or stolen devices need to be remotely wiped as
soon as possible given they may have access to customer data; speed of
reporting matters more here than in almost any other IT process. See
Device Security Requirements for what's stored locally vs. accessed only
through the VPN-gated cloud systems.

## Offboarding

Laptops must be returned within 5 business days of departure — see
Offboarding Process for the full return checklist, which includes the
laptop plus any other Northwind-issued hardware.
